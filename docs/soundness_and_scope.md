# Soundness And Scope

Status: active scope statement for Power House v0.3.23.

Power House reports must distinguish deterministic verification from broader
claims about the world.

Use:

- verifies the artifact,
- replays the lineage,
- binds semantic packets,
- rejects tampering,
- does not allocate expanded table,
- deterministic conformance replay,
- non-core semantic layer,
- portable proof memory.

Avoid:

- proves reality,
- infinite memory,
- AI is always correct,
- unbreakable,
- impossible to attack,
- solves all trust,
- replaces all science.

The default Memory Capsule soundness report states that semantic packets explain
verified state but do not change proof identity.
